# 🌿 KeenKeeper

KeenKeeper is your personal "shelf" of meaningful connections. Designed to help you nurture relationships that matter most, it allows you to track interactions, set friendship goals, and visualize your social health through insightful analytics.
Live Link: https://keen-keeper-navy.vercel.app/
---

## 🚀 Technologies Used

* **Frontend:** React.js
* **Routing:** React Router DOM
* **Styling:** Tailwind CSS
* **Icons:** Lucide-React
* **Charts:** Recharts (Data Visualization)
* **Notifications:** React Hot Toast
* **Deployment:** Vercel / Netlify

---

## ✨ Key Features

### 📊 Friendship Analytics
Visualize your social habits with a dynamic **Interactive Pie Chart**. See a breakdown of how you communicate—whether it's through Calls, Texts, or Video chats—to ensure you're balancing your interactions effectively.

### 📅 Smart Timeline & Filtering
Never lose track of a conversation again. The **Automated Timeline** logs every check-in with a timestamp and specific icon. Use the **Category Filter** to quickly toggle between different interaction types like "Calls Only" or "Recent Texts."

### ⚡ Quick Check-in System
Nurture your relationships with a single click. From the Friend Details page, you can instantly log a **Call, Text, or Video Call**. The system automatically updates the friend's status, resets the "days since contact" counter, and creates a history log.
